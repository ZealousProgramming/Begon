
# Features required:
- Account system:
	- Admin: Domain Manager
	- Distibuters: Distributes Tasks among the team
	- Resolvers: Employees who attempt to fix the issues.
- Kanban board for open, assigned, in-progress, and closed. 
- CRUD for issues
- Issues:
	- Properties:
		- Title
		- Priority
		- Status
		- Assignee
		- Reporter
		- Open Date
		- Closed Date
		- Description
		- List<Comments>
	- Create Issue
		- Submit
		- Cancel 
	- Show Details
- IssueComment:
	- Properties:
		- Author
		- Text
		- Date Posted
